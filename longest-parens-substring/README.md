# Longest Valid Parnethesis Substring

(From a prompt in the 2022-08-14 cassidoo.email.)

## Description

Given a string `s` consisting of various parenthesis `'('` and `')'`, find the length of the longest valid parenthesis substring.

### Example

```js
> parensSubstring('(()(')
> 2

> parensSubstring(')()(()))')
> 6
```
